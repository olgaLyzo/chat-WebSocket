Задание:

1)создать чат с сервером;
2)создать возможность получать текущую геолокацию;
3)учитывать code-style и best practice;

Установка и запуск проекта:

Проект собирается с помощью Vite-сборщика.
1)Скопировать url проекта: Code => Copy url to clipboard;
2)Открыть Terminal в папке <dirname>, куда будет впоследствии скачан проект, и запустить скачивание проекта => git clone <project-url>;
3)Запустить проект на локальном сервере => npm install(установить npm),
                                           npm run dev (скопировать url сервера и вставить в новое окно браузера)
4)При внесении каких-либо изменений в проект необходимо перезапустить сервер => npm run dev в Terminal и перезагрузить страницу в браузере;
5)После нажатия кнопки "Geolocation" необходимо дать согласие на поиск Вашей геолокации и немного подождать пока локация будет определяться.
